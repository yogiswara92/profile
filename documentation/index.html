Setup Mailchimp

Open "main.js" or "custom.js" file and locate the code same as below.

// Mailchimp Subscription
/* 
Notes:
=====
To get the mailchimp JSONP url (undocumented), change 'post?' to 'post-json?' and add '&c=?' to the end.
For e.g. 'http://blahblah.us1.list-manage.com/subscribe/post-json?u=5afsdhfuhdsiufdba6f8802&id=4djhfdsh99f&c=?',
*/
$('#mc-form').ajaxChimp({
  url: '//freehtml5.us15.list-manage.com/subscribe/post?u=88f804e614b4b7b925ef1b66d&id=62ecc8f57e'
});

In order to get the url value, you need to go to mailchimp.com and create an account or if you have an account already you can login there. Then, Go to "List" if you have no list yet, click "Create List" or if you have list already you can select one of those, then click "Signup forms" and select "Embedded forms" tab. After you have followed those steps you should see similar image below. Copy the highlighted code then paste it on the "url".

How To Change Style

You can change the style by adding a "style-1.css" or style-{number} up to 7. Note {number} can be upto 7 depends upon the style feature of the template. 

<link rel="stylesheet" href="css/style.css">

/* example code. */
<link rel="stylesheet" href="css/style-1.css">
